# ONNX Runtime React Native Working Example

This has been tested on Node v20.4.0

## Setup with npm
0. Install Node.js (if you haven't already)
```bash
nvm install 20
```

1. Install package dependencies
```bash
npm install
```

2. Run the app
```bash
npm run ios
```


## Setup with yarn
0. Install Node.js (if you haven't already)
```bash
nvm install 20
```

1. Install package dependencies
```bash
yarn
```

1. Prebuild
```bash
npx expo prebuild
```

2. Run the app
```bash
npm run ios
```

## Hard cleanup 
```bash
rm -rf ios/ android/ node_modules/ yarn.lock
```
