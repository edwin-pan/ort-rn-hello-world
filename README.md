# Steps of building up this project from scratch

0. Prepare environment
    1. install Node.js
    2. install expo
        ```sh
        npm install -g expo-cli
        ```
    3. install yarn
        ```sh
        npm install -g yarn
        ```

1. Setup empty project
   ```sh
   cd <SOURCE_ROOT>
   expo init . -t expo-template-blank-typescript
   ```
